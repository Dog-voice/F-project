1. USER (회원)
user_num			number		회원 등록 번호	
user_id			varchar2(100)	회원 아이디		primary key
user_pwd			varchar2(100)	회원 비밀번호
user_name		varchar2(100)	회원 이름
user_email		varchar2(100)	회원 이메일
user_zip_code		varchar2(100)	회원 우편번호
user_address		varchar2(100)	회원 주소
user_phone		varchar2(100)	회원 전화번호
user_regdate		date		회원 가입일		default sysdate
user_seq			sequence		회원 번호 생성 시퀀스	

2. PRODUCT (상품)
product_num		number		상품 등록 번호		primary key
product_name		varchar2(100)	상품 이름
product_price		number		상품 가격
product_content		varchar2(1000)	상품 설명
product_image		varchar2(100)	상품 사진	
product_regdate		date		상품 등록일		default sysdate
product_seq		sequence		상품 번호 생성 시퀀스

3. DEVELOPER (관리자)
developer_id		varchar2(100)	관리자 아이디		primary key
developer_pwd		varchar2(100)	관리자 비밀번호		

4. CART (장바구니)
cart_num			number		장바구니 번호		primary key
user_id			varchar2(100)	회원 아이디		foreign key for user_id(user)
product_num		number		상품 번호			foreign key for product_num(product) 
cart_quantity		number		장바구니에 담은 수량	
cart_result		char		배송 유무			default 'n' (배송 전 : n, 배송 후 : y)
cart_indate		date		장바구니에 담은 날짜	default sysdate
cart_seq			sequence		장바구니 번호 생성

5. ORDERS (주문하기)
orders_num		number		주문 번호			primary key
user_id			varchar2(100)	회원 아이디		foreign key for user_id(user)
orders_quantity		number		주문 수량			
orders_indate		date		주문 날짜			default sysdate
ooders_seq		sequence		주문 번호 생성		

6. ORDER_DETAIL (주문 상세)	
order_detail_num		number		주문상세 번호		primary key
orders_num		number		주문 번호			foreign key for orders_num(orders)
product_num		number		상품 등록 번호		foreign key for product_num(product)
orders_ quantity		number		주문 수량			
order_detail_result		char		주문 처리 유무		default 'n' (배송 전 : n, 배송 후 : y)
order_detail_seq		sequence		주문처리 번호 생성

7. QNA (QnA)
qna_num			number		문의 번호			primary key
qna_title			varchar2(100)	문의 제목			
qna_content		varchar2(1000)	문의 내용
qna_reply			varchar2(1000)	답변 내용
user_id			varchar2(100)	회원 아이디		foreign key for user_id(user)	
qna_rep			char		답변 유무			default 'n' (답변 전 : n, 답변 후 : y)
qna_indate		date		문의 날짜			default sysdate
qna_seq			sequence		문의 번호 생성

8. STORE (가게)
store_num		number		가게 등록 번호		
store_id			varchar2(100)	가게 아이디		primary key
store_pwd		varchar2(100)	가게 비밀번호	
store_name		varchar2(100)	가게 이름			unique
store_email		varchar2(100)	가게 이메일	
store_zip_code		varchar2(100)	가게 우편번호
store_address		varchar2(100)	가게 주소
store_phone		varchar2(100)	가게 전화번호
store_regdate		date		가게 가입일		default sysdate
store_seq			sequence		가게 번호 생성 시퀀스		

9. BOARD (게시판)
board_num		number		게시글 등록 번호		primary key
user_id			varchar2(100)	회원 아이디		foreign key for user_id(user)
store_name		varchar2(100)	가게 이름			foreign key for store_name(store)
board_category		varchar2(100)	게시글 분류 카테고리
board_title		varchar2(100)	게시글 제목
board_content		varchar2(1000)	게시글 내용
board_image		varchar2(100)	게시글 첨부 이미지
board_regdate		date		게시글 작성 날짜		default sysdate
board_sequence		sequence		게시글 번호 생성
